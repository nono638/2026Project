# Output Format Reference

All experiments produce a `raw_scores.csv` in their output directory. This document
describes every column.

## Common Columns (All Experiments)

### Query & Answer
| Column | Type | Description |
|--------|------|-------------|
| `question` | str | The query text sent to the RAG pipeline |
| `gold_answer` | str | Reference answer from the dataset (empty if no gold data) |
| `rag_answer` | str | Answer generated by the RAG pipeline |

### Extrinsic Metrics (Gold Comparison)
Only populated when gold answers exist (HotpotQA, SQuAD).

| Column | Type | Range | Description |
|--------|------|-------|-------------|
| `gold_exact_match` | bool | T/F | Does the RAG answer contain the gold answer? (case-insensitive) |
| `gold_f1` | float | 0-1 | Word-level F1 between RAG answer and gold answer |
| `gold_bertscore` | float | 0-1 | BERTScore F1 â€” semantic similarity via contextual embeddings |

### Intrinsic Metrics (LLM-as-Judge)
Scored by the configured LLM judge (default: Gemini 2.5 Flash).

| Column | Type | Range | Description |
|--------|------|-------|-------------|
| `faithfulness` | float | 0-1 | Does the answer stick to the retrieved context? |
| `relevance` | float | 0-1 | Does the answer address the question? |
| `conciseness` | float | 0-1 | Is the answer appropriately concise? |
| `quality` | float | 0-1 | Mean of faithfulness, relevance, and conciseness |

### Latency
| Column | Type | Unit | Description |
|--------|------|------|-------------|
| `strategy_latency_ms` | float | ms | Time for the RAG strategy to generate an answer |
| `scorer_latency_ms` | float | ms | Time for the LLM judge to score the answer |
| `total_latency_ms` | float | ms | Sum of strategy + scorer latency |

### Pipeline Metadata
| Column | Type | Description |
|--------|------|-------------|
| `chunk_type` | str | Chunker used (recursive, fixed, sentence, semantic) |
| `chunk_size` | int | Target chunk size in characters |
| `chunk_overlap` | int | Character overlap between chunks |
| `num_chunks` | int | Number of chunks created from the document |
| `embed_provider` | str | Embedding provider (ollama, huggingface, google) |
| `embed_model` | str | Embedding model name |
| `embed_dimension` | int | Embedding vector dimension |
| `retrieval_mode` | str | Retrieval mode (hybrid, dense, sparse) |
| `retrieval_top_k` | int | Number of chunks retrieved |
| `num_chunks_retrieved` | int | Actual chunks returned by retriever |
| `context_char_length` | int | Total character length of retrieved context |
| `reranker_model` | str/null | Reranker model name, or null if none |
| `reranker_top_k` | int/null | Chunks kept after reranking, or null |
| `llm_provider` | str | LLM provider (ollama) |
| `llm_host` | str | Ollama server URL or "local" |
| `llm_model` | str | Ollama model name (e.g., qwen3:4b) |
| `dataset_name` | str | Dataset used (hotpotqa, squad, custom) |
| `dataset_sample_seed` | int | Random seed for reproducibility |

## Experiment-Specific Columns

### Experiment 0 (Scorer Validation)
| Column | Type | Description |
|--------|------|-------------|
| `example_id` | str | Unique identifier for each query |
| `{judge_name}_faithfulness` | float | Per-judge faithfulness score |
| `{judge_name}_relevance` | float | Per-judge relevance score |
| `{judge_name}_conciseness` | float | Per-judge conciseness score |
| `{judge_name}_quality` | float | Per-judge mean quality |

Judge name prefixes: `google_gemini_2_5_flash_lite`, `google_gemini_2_5_flash`,
`google_gemini_2_5_pro`, `anthropic_claude_haiku_4_5_20251001`,
`anthropic_claude_sonnet_4_20250514`.

### Experiment 1 (Strategy x Model)
| Column | Type | Description |
|--------|------|-------------|
| `strategy` | str | RAG strategy (naive, self_rag, multi_query, corrective, adaptive) |
| `model` | str | LLM model (qwen3:0.6b, qwen3:1.7b, qwen3:4b, qwen3:8b, gemma3:1b, gemma3:4b) |

### Experiment 2 (Chunking x Model)
| Column | Type | Description |
|--------|------|-------------|
| `chunker` | str | Chunking strategy (recursive, fixed, sentence, semantic) |
| `model` | str | LLM model (qwen3:0.6b, qwen3:1.7b, qwen3:4b, qwen3:8b) |
